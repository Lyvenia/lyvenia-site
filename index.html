Lecture OBD‑II et décodage VIN

Le boîtier ELM327 lit les codes DTC, le freeze frame (RPM, charge, fuel trim au moment du défaut) et les données temps réel. Le VIN est décodé via NHTSA, complété par notre base WMI étendue, et l'IA Claude prend le relais sur les véhicules européens non reconnus.

// VIN décodé · multi-source
vin     = "VF1KJK0H541234567"
marque  = "Renault"     // NHTSA
modele  = "Kangoo"      // WMI VF1K
annee   = "2017"        // position 10
dtc     = ["P0201"]
freeze_frame: rpm=1240 · charge=68% · fuel_trim_st=+18.7%

Anamnèse en accordéon progressif

Une question à la fois, révélation en cascade. Si le véhicule ne démarre pas, RODIA saute la question « quand survient le défaut ». Description acoustique, interventions récentes, fréquence — tout est captable. Les techniciens experts peuvent tout déplier en un clic.

Surveillance moteur au ralenti

RODIA collecte un flux de mesures à 0,5 Hz pendant 60 à 180 secondes. Stats min/max/moyenne sur RPM, température, vitesse, tension batterie. Détection automatique des anomalies (RPM instables, batterie qui chute, surchauffe).

— 03 · RODIA en détail

Tout ce qu'il faut pour gérer une flotte techniquement.

Modules pensés ensemble : un diagnostic alimente l'historique, qui alimente le score, qui alimente la maintenance, qui alimente le prochain diagnostic.

Diagnostic différentiel

Suspects classés rouge / orange / jaune avec scores, causes écartées avec justification, cause racine identifiée. Pas une simple traduction de code.

Bilan de santé véhicule

4 modes : pré‑achat, périodique, pré‑contrôle technique, pré‑route longue. L'IA passe en mode préventif et fournit un plan de maintenance ciblé.

Score de fiabilité

Note 0–100 par véhicule, calculée sur statut diagnostic, alertes km, ancienneté du dernier contrôle. Tri automatique par criticité.

Calendrier de maintenance

Vidange, courroie, plaquettes, liquide de frein, contrôle technique. Statut overdue / soon / ok, états d'usure pour les pièces non programmées.

Patterns flotte

Détection automatique des codes DTC qui apparaissent sur 2 véhicules ou plus. Identifiez les défauts de série avant qu'ils ne deviennent un problème de flotte.

Rapport PDF expert

Diagnostic ou bilan exporté en PDF paginé proprement (KeepTogether, widows/orphans). Plan d'action chiffré, prêt à transmettre client ou atelier.

Détection régression km

Si le compteur affiché est inférieur au précédent diagnostic, RODIA lève une alerte fraude. Particulièrement utile en restitution leasing ou contrôle pré‑achat.

Recherche globale Ctrl+K

Command palette qui cherche véhicules, codes DTC, techniciens. Actions rapides : ouvrir un diagnostic, lancer un bilan, exporter un PDF.

Rappels NHTSA officiels

Croisement automatique avec la base des rappels constructeurs NHTSA. Si un code correspond à un rappel actif, RODIA le signale dans le diagnostic.

— 04 · Différenciateur

Une cause racine, pas une liste de suspects.

La plupart des outils OBD vous donnent une traduction du code et 5 causes possibles « par fréquence ». RODIA fait un diagnostic différentiel : il pondère chaque cause selon vos données, écarte celles qui ne tiennent pas, identifie le code secondaire d'une cause unique, et chiffre le plan d'action.

Sous le capot : Claude Opus avec un prompt structuré en 4 phases (preuves, corrélations, suspects, verdict), enrichi avec l'historique du véhicule, les réparations passées, et les rappels NHTSA.
